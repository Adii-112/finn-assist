# Fin Assist

Fin Assist is a final-year project web app built with Next.js and Tailwind CSS. It helps users manage personal finances by tracking income, expenses, budgets, and savings goals.

## Features

- Add income and expense transactions
- Categorise transactions
- Track monthly budget and savings goal
- View summary cards for income, expenses, and balance
- See spending by category
- Generate smart financial insights from transaction data
- Review recent transactions and monthly trend data
- Store data in browser localStorage for demo purposes

## Run the project

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Project idea

This application was designed as a simple personal finance assistant for students and young adults. It demonstrates CRUD-style data handling, dashboard design, budget analysis, and rule-based insight generation.
